# University Projects Portfolio🎓
This repository serves as a centralized collection of major academic projects and laboratory assignments conducted at the University of Crete. The projects span across Mathematical Seismology and Pattern Recognition, combining theoretical mathematical modeling with advanced computational implementation.

## Mathematical Seismology
These projects explores the physics of wave propagation in elastic media through numerical simulations. It bridges analytical solutions with time-domain visualizations (animation).

### Key Modules🔬
Wave Dynamics: Solving non-homogeneous wave equations in infinite and semi-infinite media using point sources and the Odd Extension method.

## Pattern Recognition
These projects focuses on the mathematical foundations of statistical learning and automated classification. It covers both Parametric and Non-Parametric methods for estimating probability density functions and defining optimal decision boundaries.

### Key Modules🔬
Statistical Parameter Estimation: Implementation of Maximum Likelihood Estimation (MLE) to determine the parameters of Gaussian distributions from training data.

Bayesian Classification: Implementing the Bayes decision rule to achieve the minimum probability of error across multi-class datasets.

Non-Parametric Density Estimation:

1. Parzen Windows: Estimating continuous density functions by interpolating data points with kernel functions (Rectangles/Gaussians).
  
2. $k$-Nearest Neighbors ($k-NN$): Implementing the $k-NN$ density estimation and classification rule, where the volume of the region is adjusted based on the distance to the $k^{th}$ nearest neighbor.

Decision Boundary Visualization: Comparative analysis of how different algorithms (Parametric vs. Non-Parametric) partition the feature space.
